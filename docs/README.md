## 🔑 Caractéristiques Principales

- 🏪 **Dashboard Restaurant** - Gestion des menus, commandes, staff
- 🛒 **Commandes Client** - Interface publique avec codes QR
- 💳 **Paiements Moneroo** - Intégration sécurisée des paiements
- 👥 **Multi-tenant** - Isolation complète des données par restaurant
- ⚡ **Real-time** - Mises à jour en temps réel avec Supabase
- 🔐 **Sécurité** - HMAC-SHA256, JWT, RLS
